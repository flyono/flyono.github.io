# Hexo的基本指令

## 快速入门

### 创建新帖子

``` bash
$ hexo new "My New Post"
```

更多信息：[写作](https://hexo.io/docs/writing.html)

### 运行服务器

``` bash
$ hexo server
```

详细信息：[服务器](https://hexo.io/docs/server.html)

### 生成静态文件

``` bash
$ hexo generate
```

详细信息：[生成](https://hexo.io/docs/generating.html)

### 部署到远程站点

``` bash
$ hexo deploy
```

详细信息：[部署](https://hexo.io/docs/one-command-deployment.html)
